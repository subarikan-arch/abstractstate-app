# Abstract State App

### The definitive directory for Manhattan’s art galleries.

## Design Philosophy
This project focuses on **minimalism and structural clarity**. The goal was to simplify the complex gallery-hopping journey into an intuitive, high-fidelity mobile experience.

## Tech Stack
- **Framework:** React Native (Expo)
- **Typography:** Cormorant Garamond (Serif), DM Mono (Monospace)
- **Styling:** Custom StyleSheet architecture with a focus on negative space and typography-driven hierarchy.

## Features
- **Dynamic Gallery Lists:** Real-time data of exhibitions in NYC.
- **Minimalist UI:** High-contrast dark mode to keep the focus on the artwork.
- **Responsive Layout:** Optimized for multiple device sizes using `Dimensions`.

## How to Run
1. Clone the repo: `git clone https://github.com/subarikan-arch/abstractstate-app.git`
2. Install dependencies: `npm install`
3. Start the project: `npx expo start`
